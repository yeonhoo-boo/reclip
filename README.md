# reclip
2025 파이썬 sw 활용 경진대회 프로젝트

온라인 강의 시청 행동 분석을 통한 중요 구간 추천해 주는 서비스입니다.

ReClipe은 온라인 강의 영상 시청 과정에서 발생하는 사용자 행동 데이터를 분석하여 긴 강의 영상 내에서 어려운 구간을 자동으로 식별하고, 복습 구간을 추천합니다.

이를 통해 복습 효율을 향상시키는 것을 목표로 한 데이터 분석 프로젝트입니다.

# **문제정의**

- 강의 중 이해가 어려웠던 구간 나중에 다시 볼 때 찾기 어려움
- 시간이 지나면 자신이 어디서 어려워 했었는지 조차 기억나지 않을 수 있음
- 복습 시 강의를 처음부터 다시 재생해야 하는 비효율
